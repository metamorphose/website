Ubuntu PPA
Allows automatically updating through Update Manager.

The personal package archive (PPA) can be added to your system manually by copying the lines below and adding them to your system's software sources.

Display sources.list entries for: 
deb http://ppa.launchpad.net/ianare/ppa/ubuntu maverick main
deb-src http://ppa.launchpad.net/ianare/ppa/ubuntu maverick main

You can also visit the PPA page in launchpad for more information.

There is an open bug requesting Métamorphose for inclusion in the official Ubuntu repos. If you have a Launchpad account, you can show that this bug affects you.
